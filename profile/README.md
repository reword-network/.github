# Reword Network

Reword Network is an open networking platform focused on high-performance
secure connectivity and infrastructure.

## Projects

- **reword-node** — Rust network engine
- **reword-control** — Control plane API
- **reword-dashboard** — Web management UI
- **reword-client-desktop** — Desktop client

## Architecture

Client → Node → Internet  
Dashboard → Control → Nodes

## Goals

- Clean networking architecture
- High performance Rust engine
- Full ecosystem: nodes, control plane, clients
